Projects Overview: Lists the projects associated with the user. Each project card displays the project title, brief description, status (e.g., In Progress, Completed), and a "View Details" button.
Search and Discovery: Offers a search bar and filters for finding new projects or mentors. Results are displayed in a list or grid format.
Notification Center: Lists recent notifications, including project approvals, new messages, and updates. Allows users to mark notifications as read or delete them.
Project Idea Submission: If the user is a student, a section allows them to submit new project ideas. Includes a form with fields for the project title, description, objectives, and file uploads.
